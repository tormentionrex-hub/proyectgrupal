¿De qué trata esta página?

Esta aplicación web fue creada con el objetivo de analizar las principales problemáticas del sistema educativo costarricense, enfocándose en la infraestructura, la gestión institucional y la administración educativa.

El proyecto busca informar, reflexionar y permitir la participación mediante comentarios y opiniones de estudiantes y docentes.

Sobre el Proyecto

Objetivo del Proyecto

El objetivo principal es desarrollar una aplicación web simple que permita visualizar problemáticas del sistema educativo costarricense usando HTML, CSS y JavaScript básico.

Motivación

Este proyecto surge como una iniciativa académica para aplicar conocimientos de programación web mientras se analiza una problemática social real.

Problemáticas Analizadas

Se abordan temas como planes de estudio desactualizados, mala infraestructura, falta de zonas recreativas y mala administración.

Impacto Esperado

Se espera generar conciencia sobre la situación educativa actual y fomentar la participación mediante opiniones y comentarios.
